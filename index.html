Buttons & Icons

Examples of button variations using classes. Change any button color by passing a $color into the mixin. Set to a border style by passing $outline:true into the mixin, or adding .button-outline to the markup. Buttons and links have different default padding and text alignment, so if you are using a mixin on the fly rather than a class on the markup (as seen below) you may need to pass $type:link into mixin. See the button mixin in _mixins.scss for more notes.
